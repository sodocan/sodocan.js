Features

More features to come. Scroll down for more info.
Easy to use CLI documentation generator

Compile from your source code, or Github repo.

It works on both commented and uncommented source code!

Clean and beautiful document templates

Use our templates to host your crowd-sourced docs for a professional look. One is available right now and more to come!

Authomatic version checking

Easy to use server API

RESTful and straight-forward API to take arbitrarily complex queries and maximum convenience for you if you were to write your own template.

Crowd-sourced documentation management

Comment, add, edit and upvote documentation entries, mark deprecated functions and add examples with ease.

Freedom to host your docs anywhere

And your docs will always be synced to our database. If someone else hosts docs for the same library, your versions will be synced up.

sodocan-parser
